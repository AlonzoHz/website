Description

Ear Book was my first PHP project, started in spring 2015. I joked with some friends that instead of buying a yearbook I would make my an earbook that contains pictures of people's ears. One day I was bored and actually started working on it. Although it never extended beyond my high school, it saw a great deal of popularity over the summer, with over 4000 votes and 183 unique viewers (versus 200 in my graduating class).

What I Learned

This project was the first time that I used PHP and gave me the skills to do other projects in PHP. I also learned about website formatting and file structures that aid development. Unfortunately, I lost the SQL database associated with the page, so all of the stats are gone. Needless to say, I have been much better about making backups since then.

Future

Although this project is long since over, I plan to generalize it as a webpage for comparing and ranking images.
