Description

Earbook was my first PHP project, started in spring 2015. I joked with some friends that instead of buying a yearbook I would make an earbook showing only pictures of people's ears. One day I was bored and actually started working on it. I realized that we all could have some fun if site visitors could send me a picture of their own ear and participate in side-by-side voting to rank the "best ear in the school."

Although its notoriety never extended beyond my classmates in high school, the website was very popular over the summer, gathering over 4400 votes and 183 unique viewers. Considering that there were only 200 students in my graduating class, I think this project was a surprising success!

What I Learned

This project was the first time that I used PHP and gave me the skills to do other projects in PHP. I also learned about website formatting and file structures that aid development. Unfortunately, I lost the SQL database associated with the page, so all of the stats are gone. Needless to say, I have been much better about making backups since then.

Future

Although this project is long since over, I plan to generalize it as a webpage for comparing and ranking images.
