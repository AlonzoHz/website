Description

Dress Me Up is a PHP and SQL webpage that I wrote in later summer 2016 to help my girlfriend tell me what I should wear on dates. The interactive page takes pieces of clothing I've drawn and indexed in an SQL database and makes them available for user selection based on their type.

What I Learned

Because this project involves layering images, creating it required me to read extensively about how webpage elements are positioned and displayed relative to each other on the screen. In addition, I had to figure out how to create an iteratively populated graphical menu of the sort used to show all of the clothing options.

Future

At this point, the code is functional, but I need to add more clothing. I also want to add a submission system so that I can be informed when someone wants to wear me something to an event.
